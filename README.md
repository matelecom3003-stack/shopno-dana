## কীভাবে শুরু করবেন (Getting Started)

প্রজেক্টটি আপনার কম্পিউটারে চালাতে নিচের ধাপগুলো অনুসরণ করুন:

১. রিপোজিটরি ক্লোন করুন:
git clone https://github.com/matelecom3003-stack/shopno-dana.git

২. প্রজেক্ট ফোল্ডারে যান:
cd shopno-dana

৩. রান করুন:
python app.py
